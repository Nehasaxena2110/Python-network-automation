# YAML Parsing Examples

This folder contains examples of reading and parsing YAML files using Python.

## Topics Covered

- YAML Basics
- Device Inventories
- YAML Parsing with PyYAML
- Automation Data Structures

## Files

| File | Purpose |
|--------|---------|
| devices.yaml | Sample network inventory |
| read_yaml.py | Read and display device information |

## Requirements

```bash
pip install pyyaml
```
